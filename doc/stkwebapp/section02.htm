2.1 StkWebAppの構成

StkWebAppは，CGIコンポーネント，WebApp ServiceコンポーネントおよびJavaScript libraryで構成される。
WebApp Serviceコンポーネントは，StkSocket，StkThread，StkDataを内部に梱包している。

CGIコンポーネント，WebApp ServiceコンポーネントおよびJavaScript libraryの提供形態は下記のとおり。
コンポーネント提供形態備考
CGI.exeファイルHTTP ServerとWebApp Service間を中継する。
WebApp Service.libファイル StkSocket，StkThread，StkDataを内部に梱包したWebアプリケーションの制御に特化したライブラリ。 WebApp Service固有の機能として，JSON/XMLと内部オブジェクトStkObjectを相互変換する機能（StkObjectConverter）， リクエストを適切な処理オブジェクトにディスパッチする機能(StkReqHandler)を有する。
JavaScript library.jsファイルクライアント側のリクエスト／レスポンス処理ライブラリ。
