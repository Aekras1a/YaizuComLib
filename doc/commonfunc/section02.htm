2. MessageProc

2.1 MessageProc概要

MessageProcは，クライアントプログラムのメッセージリソースおよびラベルリソースの管理を支援するクラスです。
MessageProcは，主に以下の機能をクライアントプログラムに提供します。
  • メッセージリソースおよびラベルリソースの保持（最大10,000件）
  • ロケールに応じたメッセージリソースおよびラベルリソースの取得（英語ロケール，日本語ロケールに対応）
  • 指定したエンコード種別に応じたメッセージリソースおよびラベルリソースの取得（UTF-16, UTF-8, S-JISに対応）

2.2 MessageProcの使用方法

MessageProcの機能を利用するために，クライアントプログラムは，以下の.hファイル，.libファイルを事前に参照可能な状態にしておく必要があります。

.hファイルmsgproc.h
.libファイルcommonfunc.lib

MessageProcのすべてのAPIはstatic宣言されており，クライアントプログラムから下記のように呼び出すことができます。

ex.) MessageProc::GetMsg(100);

2.3 API一覧

MessageProcクラスが提供するAPI一覧を以下に示します。

static void SetLocaleMode(int)GetMsg, GetMsgSjis, GetMsgUft8 APIが返すメッセージのロケールを指定します。
static TCHAR* GetMsg(int)UTF-16エンコーディングしたメッセージを返却します。
static TCHAR* GetMsgEng(int)UTF-16エンコーディングした英語メッセージを返却します。
static TCHAR* GetMsgJpn(int)UTF-16エンコーディングした日本語メッセージを返却します。
static BYTE* GetMsgSjis(int)SJISエンコーディングしたメッセージを返却します。
static BYTE* GetMsgSjisEng(int)SJISエンコーディングした英語メッセージを返却します。
static BYTE* GetMsgSjisJpn(int)SJISエンコーディングした日本語メッセージを返却します。
static BYTE* GetMsgUtf8(int)UTF-8エンコーディングしたメッセージを返却します。
static BYTE* GetMsgUtf8Eng(int)UTF-8エンコーディングした英語メッセージを返却します。
static BYTE* GetMsgUtf8Jpn(int)UTF-8エンコーディングした日本語メッセージを返却します。
static void AddEng(int, TCHAR*)指定した英語メッセージを追加します。
static void AddJpn(int, TCHAR*)指定した日本語メッセージを追加します。
static void DelEng(int)指定した英語メッセージを削除します。
static void DelJpn(int)指定した日本語メッセージを削除します。
static void ClearAllMsg()すべてのメッセージを削除します。
